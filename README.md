# 🥘 Sabor Andino - Delivery App

![Kotlin](https://img.shields.io/badge/kotlin-%237F52FF.svg?style=for-the-badge&logo=kotlin&logoColor=white)
![Android](https://img.shields.io/badge/Android-3DDC84?style=for-the-badge&logo=android&logoColor=white)
![Jetpack Compose](https://img.shields.io/badge/Jetpack%20Compose-4285F4?style=for-the-badge&logo=jetpackcompose&logoColor=white)

**Sabor Andino** es una aplicación móvil moderna desarrollada en Android para la gestión de pedidos de un restaurante de comida tradicional. La aplicación ofrece una experiencia fluida y visualmente atractiva utilizando las últimas tecnologías de desarrollo nativo.

---

## ✨ Características

- 🔐 **Autenticación:** Pantalla de login personalizada.
- 🏠 **Home interactivo:** Bienvenida personalizada al usuario.
- 📜 **Menú Digital:** Exploración de platos típicos con imágenes y descripciones.
- 🛒 **Gestión de Pedidos:** Selección de platos y carrito de compras (en desarrollo).
- 📱 **Diseño Moderno:** Interfaz construida íntegramente con **Jetpack Compose** y **Material 3**.
- 🧭 **Navegación Fluida:** Implementación de *Type-safe Navigation*.

---

## 🛠️ Tecnologías Utilizadas

- **Lenguaje:** [Kotlin](https://kotlinlang.org/)
- **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Arquitectura:** MVVM (Model-View-ViewModel)
- **Componentes:**
    - Material 3 para el diseño visual.
    - Navigation Compose para el flujo entre pantallas.
    - ViewModel & State management.

---

## 📁 Estructura del Proyecto

```text
com.tecsup.sabor_andino/
├── data/          # Repositorios y Modelos de datos
├── navigation/    # Configuración de rutas y NavGraph
├── screens/       # Pantallas de la UI (Composables)
├── viewmodel/     # Lógica de negocio y estado de la UI
└── MainActivity.kt # Punto de entrada de la aplicación
```

## 🚀 Cómo empezar

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/sabor-andino.git
   ```
2. **Abrir en Android Studio:**
   Asegúrate de tener la última versión estable de Android Studio (Iguana o superior).
3. **Ejecutar:**
   Selecciona un emulador o dispositivo físico y presiona `Run`.
